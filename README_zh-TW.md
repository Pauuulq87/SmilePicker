# SmilePicker

<img src="https://api.iconify.design/mdi/emoticon-outline.svg" width="64" height="64" alt="SmilePicker"/>

Chrome 側邊面板符號快速輸入工具。快捷鍵一觸即達，智慧搜尋、收藏同步、深色模式。

## 功能特色

- **即時存取** — 快捷鍵 `Alt+S` 一觸即達，側邊滑出不打斷工作流程
- **精緻體驗** — 現代化 UI 設計，視覺愉悅
- **極致效率** — 智慧搜尋、鍵盤導航、一鍵複製
- **無縫同步** — Chrome 帳號同步收藏與設定

## 符號庫

| 裝飾符號 | ★ ♥ ● ○ |

## 技術棧

- TypeScript + React 18
- Tailwind CSS
- Vite + CRXJS
- Chrome Extension Manifest V3

## 開發

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 建置
npm run build
```

## 載入擴充功能

1. 開啟 `chrome://extensions/`
2. 啟用「開發人員模式」
3. 點擊「載入未封裝項目」
4. 選擇 `dist/` 資料夾
