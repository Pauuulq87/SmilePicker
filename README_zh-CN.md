# SmilePicker

<img src="https://api.iconify.design/mdi/emoticon-outline.svg" width="64" height="64" alt="SmilePicker"/>

Chrome 侧边面板符号快速输入工具。快捷键一触即达，智能搜索、收藏同步、深色模式。

## 功能特色

- **即时访问** — 快捷键 `Alt+S` 一触即达，侧边滑出不打断工作流程
- **精致体验** — 现代化 UI 设计，视觉愉悦
- **极致效率** — 智能搜索、键盘导航、一键复制
- **无缝同步** — Chrome 账号同步收藏与设置

## 符号库

| 类别 | 示例 |
| --- | --- |
| 箭头符号 | → ← ↑ ↓ ⇒ |
| 数学符号 | ± × ÷ ≠ ∞ |
| 标点符号 | 「」『』【】 |
| 键盘符号 | ⌘ ⌥ ⇧ ⌃ |
| 货币符号 | ¥ € £ ₿ |
| 装饰符号 | ★ ♥ ● ○ |

## 技术栈

- TypeScript + React 18
- Tailwind CSS
- Vite + CRXJS
- Chrome Extension Manifest V3

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 加载扩展

1. 打开 `chrome://extensions/`
2. 启用「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `dist/` 文件夹
