# SmilePicker

<img src="https://api.iconify.design/mdi/emoticon-outline.svg" width="64" height="64" alt="SmilePicker"/>

Chrome side panel for quick symbol input. Hotkey access, smart search, cloud sync, dark mode.

## Features

- **Instant Access** — `Alt+S` hotkey, slides out without interrupting workflow
- **Polished UX** — Modern UI design, visually pleasing
- **Maximum Efficiency** — Smart search, keyboard navigation, one-click copy
- **Seamless Sync** — Chrome account syncs favorites and settings

## Symbol Library

| Category | Examples |
| --- | --- |
| Arrows | → ← ↑ ↓ ⇒ |
| Math | ± × ÷ ≠ ∞ |
| Punctuation | 「」『』【】 |
| Keyboard | ⌘ ⌥ ⇧ ⌃ |
| Currency | ¥ € £ ₿ |
| Decorative | ★ ♥ ● ○ |

## Tech Stack

- TypeScript + React 18
- Tailwind CSS
- Vite + CRXJS
- Chrome Extension Manifest V3

## Development

```bash
# Install dependencies
npm install

# Dev mode
npm run dev

# Build
npm run build
```

## Load Extension

1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `dist/` folder
